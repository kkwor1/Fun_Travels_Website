# Fun Travells - Kyrgyzstan Travel Agency Website

Welcome to the repository for **Fun Travells**, a modern and responsive travel agency website dedicated to adventurous tours in Kyrgyzstan.

## 🏔️ About the Project

This project is a beautifully designed, front-end landing page for a fictional travel agency, "Fun Travells". It highlights majestic mountains, nomadic culture, and alpine lakes through curated tours. The website features a clean layout, stunning imagery, and a smooth user experience to inspire travelers to book their next adventure.

## ✨ Key Features

- **Modern & Responsive Design:** Fully responsive layout that looks great on desktops, tablets, and mobile devices.
- **Interactive UI:** Smooth scrolling, parallax hero background, and fade-in animations on scroll.
- **Bilingual Support (English / Russian):** Built-in language toggle functionality for switching content dynamically.
- **Tours Display:** A curated grid showcasing featured tours like Winter Alpine Retreats and Nomadic Yurt Heritage.
- **Booking Modal:** An accessible, glassmorphism-styled booking form overlay.
- **Dynamic Statistics:** Animated number counters highlighting company experience and traveler satisfaction.
- **Image Gallery:** A visually appealing section showcasing moments captured during expeditions.

## 🛠️ Technologies Used

- **HTML5:** Semantic markup for structure.
- **CSS3:** Custom styling including Flexbox, CSS Grid, animations, and glassmorphism effects.
- **JavaScript (Vanilla):** Logic for the mobile menu, language toggling, modal interactions, and scroll animations.
- **Lucide Icons:** Clean and consistent iconography.
- **Google Fonts:** Utilizing 'Montserrat' for headings and 'Open Sans' for body text.

## 🚀 Getting Started

To view the website locally, no build tools or package managers are required.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/kkwor1/Fun_Travels_Website.git
   ```
2. **Navigate to the directory:**
   ```bash
   cd Fun_Travels_Website
   ```
3. **Open `index.html`:**
   Simply open the `index.html` file in your preferred web browser. Alternatively, you can use a local server extension like Live Server in VS Code for a better development experience.
